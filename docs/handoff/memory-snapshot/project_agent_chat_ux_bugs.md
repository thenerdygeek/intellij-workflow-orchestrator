**Root causes (file:line):**
1. **History resume auto-iterates / can't continue** ‚Äî `AgentService.resumeSession` only short-circuits the loop for completed sessions (`AgentService.kt:2863` `RESUME_COMPLETED_TASK`); interrupted sessions unconditionally `executeTask(finalPreamble)` (`:3163`).
