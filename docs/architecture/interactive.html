Modules never import each other. They communicate through a central SharedFlow event bus in :core. Press Play to watch events flow.

Service Layer Architecture

Every service returns ToolResult<T> with typed .data for UI panels and a text .summary for logging and notifications. Click Animate to trace a request from UI through the service layer to the external API and back.

Threading Model

Three thread domains: EDT (UI only, 16ms budget), IO (network/disk), and Background (polling/tasks). Select a scenario to watch dots flow between lanes.
